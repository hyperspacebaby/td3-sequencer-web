⚡ TD-3 MIDI SNIFFER

Connect your TD-3 as INPUT · press buttons on the hardware · see the exact raw bytes here.

HOW TO USE
  1. 1. Select your TD-3 from the INPUT dropdown below and click CONNECT
  2. 2. Put the TD-3 into TIME MODE (press TIME MODE button, LED blinks)
  3. 3. Press the NOTE button (Step key) once → read the hex
  4. 4. Press TIME MODE again to reset, then press TIE (Write/Next) → read the hex
  5. 5. Press TIME MODE again, press REST (Back) → read the hex
  6. 6. The "CAPTURED" box below auto-fills — share those 3 bytes with Claude
